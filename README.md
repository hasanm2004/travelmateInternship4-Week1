# TravelMate ✈️

TravelMate is a modern travel landing page designed to help users explore travel services and plan their next adventure. The project was developed using pure HTML, CSS and JavaScript without any frameworks.

## Features

- Responsive Design
- Sticky Navigation Bar
- Smooth Scrolling CTA Button
- Feature Cards
- About Section
- Travel Tools Section
- Statistics Section
- Testimonials
- Contact Form
- Back To Top Button
- Mobile Friendly Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
